Installing Postgres
===================

For Mac
~~~~~~~

`Postgres App <http://www.postgresapp.com>`_

Or 

.. code-block:: sh

   brew install postgresql


For Ubuntu
~~~~~~~~~~

.. code-block:: sh

   sudo apt-get install postgresql

For Windows
~~~~~~~~~~~


`Windows Installer <http://www.enterprisedb.com/products-services-training/pgdownload#windows>`_
